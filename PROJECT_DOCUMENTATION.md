# RewardVerse Landing Page Documentation

## Project Overview
RewardVerse is a premium, high-conversion landing page for the RewardVerse Native Android Application. It is designed to drive app downloads and build trust with a modern, mobile-first aesthetic inspired by top consumer apps like CRED and Duolingo.

## Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Fonts**: Sora (Headings), Inter (Body)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theme**: Premium Dark Android App Theme (#070B14)

## Design System
- **Colors**:
  - Background: `#070B14`
  - Primary Purple: `#8B5CF6`
  - Accent Pink: `#EC4899`
  - Accent Cyan: `#22D3EE`
  - Text: `#F8FAFC`
- **Typography**:
  - Headings: `Sora` (Futuristic, clean, app-native)
  - Body: `Inter` (Professional, readable)
- **Assets**:
  - Branding: `logo.png` integrated into Navbar, Hero, and Favicon.
- **Layout**:
  - High vertical spacing (`py-32`)
  - Soft shadows and subtle glassmorphism
  - Minimalist borders

## Page Structure

### 1. Navbar
- Minimalist conversion-focused bar with RewardVerse Logo and "Play Store" CTA.

### 2. Hero Section
- 90vh section with an Android Phone Mockup.
- Floating "Earning Proof" notifications (Payouts, Bonuses).
- Primary CTA: "Play Store" (Link to Google Play).

### 3. Trusted Stats
- Metrics focused on app success: 50k+ Downloads, ₹5L+ Rewards, 100k+ Tasks, 4.8 Rating.

### 4. App Preview (Screenshots)
- Visual carousel showcasing the actual Android app interface (Home, Tasks, Wallet, Refer).

### 5. Features
- 6 Key app features: Surveys, Offerwalls, Referrals, Instant Payouts, Daily Bonus, and Gift Cards.

### 6. How It Works
- Simplified 3-step app flow: Download from Play Store -> Complete Tasks -> Redeem Cash.

### 7. Rewards Showcase
- Premium floating cards for major redemption partners: Amazon, Paytm, Google Play, and UPI.

### 8. Testimonials
- Play Store-style reviews with star ratings and verified earner status.

### 9. Download CTA
- Large gradient banner with Play Store focus.

### 10. FAQ & Footer
- 5 Core questions about the Android app.
- Minimalist footer with RewardVerse branding and legal pages.
